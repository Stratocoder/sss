Super Simple Stocks
===================

Implementation of Super Simple Stocks task written in Python by Yurii
Konovaliuk.

How to
------

This implementation would work on any platform, although out of box scripts are
witten for \*nix systems.

*   To run tests:

    $ make test

*   To run interactive shell:

    $ ./sss/sss.py
